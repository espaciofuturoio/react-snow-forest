# React Native Skia - Snow Forest

I have created an animation using React Native Skia and Matter.js as a 2D rigid body JavaScript physics engine.

## Setup project

1. Install dependencies

   ```bash
   bun i
   ```

2. Start the app

   ```bash
   bun run web
   ```
